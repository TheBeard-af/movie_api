Welcome to My Movies Web Application

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

API Documentation

Get All Movies

URL: /movies

Method: GET

Description: Returns a list of all movies.

Response Format:

[{
        "Title": "The Matrix",
        "Description": "A dystopian future with advanced AI.",
        "Genre": { "Name": "action", "Description": "Shooting and stuff" },
        "Director": { "Name": "Lana Wachowski", "Bio": "Visionary filmmaker.", "Birth": 1965 }
    }]

Get Movie by Title

URL: /movies/:title

Method: GET

Description: Returns details of a specific movie by its title.

Parameters: title (string)

Response Format:

{
        "Title": "The Matrix",
        "Description": "A dystopian future with advanced AI.",
        "Genre": { "Name": "action", "Description": "Shooting and stuff" },
        "Director": { "Name": "Lana Wachowski", "Bio": "Visionary filmmaker.", "Birth": 1965 }
    }

Get Movies by Genre

URL: /movies/genre/:genreName

Method: GET

Description: Returns movies belonging to a specific genre.

Parameters: genreName (string)

Response Format:

[{
        "Title": "The Matrix",
        "Description": "A dystopian future with advanced AI.",
        "Genre": { "Name": "action", "Description": "Shooting and stuff" },
        "Director": { "Name": "Lana Wachowski", "Bio": "Visionary filmmaker.", "Birth": 1965 }
    }]

Get Director Details

URL: /movies/directors/:directorName

Method: GET

Description: Returns details of a director by their name.

Parameters: directorName (string)

Response Format:

{
        "Name": "Lana Wachowski",
        "Bio": "Visionary filmmaker.",
        "Birth": 1965
    }

Create a User

URL: /user

Method: POST

Description: Creates a new user.

Request Body:

{ "name": "Kim" }

Response Format:

{ "id": "uuid", "name": "Kim", "favouriteMovies": [] }

Update a User

URL: /user/:id

Method: PUT

Description: Updates a user's details.

Parameters: id (string)

Request Body:

{ "name": "New Name" }

Response Format:

{ "id": "uuid", "name": "New Name", "favouriteMovies": [] }

Add Movie to User's Favourites

URL: /user/:id/:movieTitle

Method: POST

Description: Adds a movie to a user's favourites list.

Parameters: id (string), movieTitle (string)

Response Format:

"{movieTitle} has been added to user {id}'s array"

Remove Movie from User's Favourites

URL: /user/:id/:movieTitle

Method: DELETE

Description: Removes a movie from a user's favourites list.

Parameters: id (string), movieTitle (string)

Response Format:

"{movieTitle} has been removed from user {id}'s array"

Delete a User

URL: /user/:id

Method: DELETE

Description: Deletes a user by their ID.

Parameters: id (string)

Response Format:

"User {id} has been deleted"
